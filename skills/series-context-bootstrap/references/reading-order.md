# Reading Order

Follow this order unless the user overrides it.

## 1. Canon Entry

Read `canon/world-setting.md` first.

It is an index. Use it to identify which files are relevant to the task before reading further.

## 2. Topic Canon

Read only the numbered canon files relevant to the task.

File mapping for this repository:

- world structure, technology, gods, organizations: `canon/02_世界観設定資料.md`
- MPHTSU unit types and specs: `canon/01_MPHTSU-2166_系列一覧.md`
- character — Artemisia: `canon/04_アルテミシア_人物設定資料.md`
- character — Nadia Gromova: `canon/03_ナディア・グロモワ_人物設定資料.md`
- character — Claudia: `canon/05_クラウディア_人物設定資料.md`
- arc overview, Part 1 theme and incident axis: `canon/06_第一部_アーク概要.md`

## 3. Current Story Position

To locate present narrative state, prefer:

1. the latest files in `posts/`
2. the current related files in `drafts/`
3. flow or plot notes for the target episode

Read full episode text only when scene-level carryover matters.

## 4. Local Carryover

If the task is about the next episode or a revision near the front of the current arc, read:

1. the immediately previous published episode in full
2. the previous one before that only if emotional carryover or unresolved action requires it

Current draft state (as of 2026-04-19):
- Episode 1 draft: `drafts/01_第一話_労働局へようこそ.md`
- Episode 2 draft: `drafts/02_第二話_硝子越しの声.md` (incomplete — interrogation scene missing)
- No published posts yet

## 5. Stop Rule

Stop reading as soon as you can answer:

- where the story currently is
- what canon matters for this task
- what the recent emotional and tactical carryover is
- what files should be handed to the next workflow

Do not keep reading just because more files exist.
