# Draft Shaping

Write scenes, not disguised outlines.

## Preferred Movement

1. Carry over the prior episode's emotional temperature
2. Establish the present task or situation quickly
3. Let character interaction do the main work
4. Introduce one pressure from the wider world
5. End with either:
   - a next task
   - a changed relationship
   - a sharpened unresolved problem

Do not force all three unless the episode truly supports it.

## Prefer

- action and reaction over explanation
- short, functional dialogue
- worldbuilding revealed by situation
- emotional movement through pause, phrasing, and behavior

## Avoid

- canon dumps in dialogue
- over-resolving tension that should carry forward
- making every episode climax-heavy
- overpolishing if the user wants a draft to revise themselves

## Author Enrichment

When useful, leave room for the user to intensify:

- a key line of dialogue
- one strong image
- a transition paragraph
- the closing cadence
