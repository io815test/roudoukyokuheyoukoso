# Episode Function

Define the episode's job before writing prose.

## Lock These Points

- what this episode must accomplish
- which character relationship changes here
- what the next episode should inherit

## Limits

- prefer one primary function
- allow at most one secondary function
- do not try to advance every thread in one episode

## Good Examples

- convert a rescue incident into a durable bond
- transition from aftermath into a new field assignment
- introduce one new institutional or world pressure while keeping character focus

## Bad Examples

- deepen everyone
- advance every subplot
- explain more lore because there is room

## Check Before Drafting

- does this episode change anything real
- does it connect to the previous episode emotionally
- does it hand something concrete to the next episode
