# Post Draft Update

After drafting, write a compact update that can be reused as future context.

## Include

- 3-5 line summary of the episode
- character-state changes
- new unresolved threads
- canon-sensitive additions introduced here
- what the next episode now needs to do

## Rules

- keep it short
- prefer direct facts over prose summary
- write for future reuse, not for publication
- update the memory files if the project maintains them
